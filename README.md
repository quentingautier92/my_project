# P2018 A3 Gr3 - Symfony

## Install the project

```
git clone git@github.com:Irvyne/P2018_A3_Gr3_Symfony.git
```

```
cd P2018_A3_Gr3_Symfony
```

```
composer install
```

```
php app/console doctrine:database:create
```

```
php app/console doctrine:schema:update --dump-sql --force
```
